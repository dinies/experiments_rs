## experiments_rs
A bunch of experiments in Rust.
At the moment, the main executable is a graphical simulation of Conway game of
life.
And, in the examples, there is a collection of problems taken from LeetCode.

### How to run the code
- Make sure rust is installed
- To run the main executable: `cargo run --release`
- To run the examples: `cargo run --release --example example_name`
